1．项目概况

该教程适用的项目是使用vue脚手架vue-cli搭建，后端使用express，数据库使用的是Mysql的项目。项目最终部署至阿里云ECS云服务器上。阿里云ECS服务器安装的系统为centos 7。

2．部署流程

2.1 实用工具安装Xshell，Xftp，Navicat

  • Xshell: 远程连接登录阿里云ECS服务器命令行工具
  • Xftp: 可像管理windows文件夹般管理阿里云ECS服务器中的文件夹，对于不熟悉linux文件操作命令的同学很有帮助

  • Nacvicat: 远程连接ECS服务器中安装的MySQL，远程建库建表，进行CRUD等操作
